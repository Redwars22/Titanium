A minimalistic experimental interpreted programming language!

Titanium's goal was initially to help me understand how programming languages work and are built, but you can also try it out and write your first programs in Titanium too! You can download AndrewToolchain and run Titanium source code files in your PC. Please read the instructions in its repository to download, compile and run it in your machine.

What is Titanium?

Minimalistic

Titanium was designed to have an easy-to-read and clean syntax and to be easy to learn

Open-source

You can freely download, read, and modify Titanium's source code. Wanna add new features? Then go for it!

Portable

Titanium was written in TypeScript, which ensures that it can be compilled, adapted, and ported to run virtually anywhere.
